# Primero Paint & Drywall — primeropaint.com

Static HTML site. Deploy via Vercel or any static host.

## Deploy to Vercel

1. Push this repo to GitHub
2. Import from vercel.com → New Project → select repo
3. Framework: **Other** (no build step needed)
4. Output directory: leave blank (root)
5. Click Deploy

## Custom Domain

In Vercel dashboard → Settings → Domains → add `primeropaint.com`

## Update Phone / Email

Search `index.html` for:
- `tel:+19015550000` → replace with real number
- `hello@primeropaint.com` → replace with real email
